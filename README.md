# Как запустить чат с ботом:
## 1. В терминале git clone https://github.com/OsperGiT/MospolytechChad
## 2. В терминале: cd [ПАПКА С ПРОЕКТОМ]
## 3. В терминале: python -m venv venv
## 4. Установить зависимости: pip install -r requirements.txt
## 5. Установить ollama: https://ollama.com/download
## 6. В терминале: ollama push gemma3:12b
## 7. Запусти chat.py
## 8. Включить/выключить отладку в chat.py меняй значение переменной otladka
